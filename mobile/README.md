# Premium ChatGPT React Native App

This is a modern, high-fidelity ChatGPT interface built with React Native and Expo.

## Features

- **Midnight Theme**: Sleek dark mode design.
- **Glassmorphism & Gradients**: Premium visual effects using `expo-linear-gradient`.
- **Typing Simulation**: Realistic "thinking" state.
- **Responsive Layout**: Optimized for iOS and Android.
- **Lucide Icons**: Modern icon set for a clean look.

## Getting Started

1.  **Navigate to the mobile directory**:
    ```bash
    cd mobile
    ```

2.  **Start the development server**:
    ```bash
    npx expo start
    ```

3.  **Open the app**:
    - Scan the QR code with the **Expo Go** app (Android/iOS).
    - Press `i` for iOS simulator.
    - Press `a` for Android emulator.

## Dependencies

- `expo`: Framework
- `react-native`: Core
- `expo-linear-gradient`: Gradients
- `lucide-react-native`: Icons
- `react-native-safe-area-context`: Layout safety

## Project Structure

- `src/components/ChatInterface.tsx`: Main UI and logic.
- `src/types.ts`: TypeScript interfaces.
- `App.tsx`: App entry point.
